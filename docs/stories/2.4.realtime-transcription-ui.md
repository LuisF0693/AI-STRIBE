## Story
**As a** médico,
**I want** ver a transcrição da consulta sendo exibida em tempo real enquanto atendo o paciente,
**so that** eu possa acompanhar o que está sendo captado e ter confiança de que o sistema está funcionando corretamente.

## Acceptance Criteria
1. Tela "Consulta Ativa" exibe texto da transcrição sendo construído em tempo real (streaming via Supabase Realtime)
2. Texto exibido com diferenciação visual entre falas do médico e do paciente (quando detectável)
3. Latência visual máxima de 2 segundos entre a fala e o texto aparecer na tela (NFR2)
4. Tela permanece responsiva durante gravação — rolagem suave sem travamentos
5. Indicador de "qualidade de áudio" (ícone de microfone com barra de nível) visível durante gravação
6. Botão de pausa/retomada de gravação acessível sem sair da tela de transcrição
7. Tela de "Processando nota..." exibida após encerramento da consulta com estimativa de tempo restante
8. Notificação in-app quando nota SOAP estiver pronta (sem precisar manter o app em foreground)
9. Layout acessível: fonte mínima 16sp, contraste WCAG AA, suporte a Dynamic Type (iOS) e Font Scale (Android)
10. Funciona em modo landscape e portrait sem quebrar layout

## Dev Notes

### Contexto Técnico
- Dependências: Story 2.1 (gravação iniciada), Story 2.2 (transcrição em progresso via Realtime)
- Supabase Realtime: subscriber no canal `realtime:public:transcricoes:consulta_id=eq.{id}`
- Expo Notifications: configurar em Story 1.4 (onboarding) — esta story usa a configuração existente

### Performance
- `FlatList` com `initialNumToRender={10}` e `maxToRenderPerBatch={5}` para evitar jank
- Evitar re-renders desnecessários: usar `React.memo` nos componentes de segmento
- Subscription Realtime gera atualizações frequentes — usar `useCallback` e `useMemo` onde aplicável

### Design Reference
- Tela de gravação: fundo branco, texto preto, indicador de gravação em vermelho (#E53935)
- Font: System font — SF Pro no iOS, Roboto no Android (sem fonte customizada para performance)
- Balão médico: fundo azul claro (#E3F2FD); balão paciente: fundo cinza claro (#F5F5F5)

### Testing
- Localização: `apps/mobile/src/screens/ConsultaAtiva/__tests__/`
- Framework: React Native Testing Library + Jest
- Testar: renderização de 100+ segmentos sem jank, comportamento de pausa/retomada
- Device físico obrigatório para testes de orientação e performance real
